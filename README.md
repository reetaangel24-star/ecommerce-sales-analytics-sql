# E-Commerce Sales Analytics Dashboard

## Project Overview

This project demonstrates real-world business analytics using MySQL and e-commerce transaction data.

### Objectives

- Analyze sales performance
- Identify top-selling products
- Segment customers
- Track monthly revenue trends
- Measure repeat customer behavior
- Monitor inventory performance
- Build business reports using SQL

### Technologies Used

- MySQL
- SQL
- Git
- GitHub

### SQL Concepts Covered

- Joins
- CTEs
- Window Functions
- Views
- Stored Procedures
- Triggers
- Aggregations
- Ranking Functions
